The Chrome debugging protocol allows for tools to instrument, inspect, debug and profile Chromium, Chrome and other Blink-based browsers. The Chrome DevTools uses this protocol and the team maintains its API.

For production use, consider the stable protocol version. Many existing projects currently use the protocol.
